## 🧭 My Positioning

**Scenario → Constraint → Agent**

- **Scenario:** freight-searching, campus delivery, vehicle routing, time windows, user preferences, and operational bottlenecks.
- **Constraint:** translate business rules, uncertainty, capacity, energy, risk, and natural-language preferences into models, rules, scoring, and validation gates.
- **Agent:** use LLM workflows, prompt engineering, tool use, state tracking, and full-stack demos to make decision support interactive and inspectable.
